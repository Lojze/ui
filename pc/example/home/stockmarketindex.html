股指

MMIG NZ——股指投资者的最佳平台选择
股指即股票差价合约是一种以保证金作交易的股票衍生产品。由于股票价格起伏无常， 对于具体某一种股票的价格变化，投资者容易了解，而对于多种股票的价格变化，要逐一了解，既不容易，也不胜其烦。因此，产生了股票指数，该指数是一组特定的股票的平均价值，被公开发布，作为市场价格变动的指标。投资者据此就可以检验自己投资的效果，并用以预测股票市场的动向。

  • 为什么选择交易指数？

    • 低投入，高杠杆
    • 免佣金
    • 风险对冲
    • T + 0 交易模式
    • 双向交易，可做多做空

  • 低投入，高杠杆

    不同于传统的股票市场，在我们的平台交易同样可以享受到我们的另一优势——交易杠杆，股指杠杆更高达100倍。

    免佣金

    股票投资及股指期货均需支付交易佣金，而XXXFX提供的各种中国及国外股指产品则免交易佣金，减省投资成本 。

    T + 0 交易模式

    股票投资一般进行T + 1交易模式，买入卖出时间受限制。通过T + 0 交易模式，投资股指没有买入卖出时间限制，便于双向捕捉机会。

    双向交易，可做多做空

    投资者通过做多做空，股市涨跌皆有机会获利，无惧跌市。

    风险对冲
